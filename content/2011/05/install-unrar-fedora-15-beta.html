--- title: Install unrar Fedora 15 beta created: !!timestamp '2011-05-02 00:00:00' tags: - unrar - fedora --- {% mark excerpt %} Unrar has yet to come to the repo, but its super easy to install. 1. Download the binary from their site. http://www.rarlab.com/rar/unrar-linux-x64.gz 2. gunzip the file 3. Copy it to /usr/bin as root 4. Give it the correct permissions 0755 Heres a simple script that will do all that for you. #!/bin/bash # Install unrar binary to /usr/bin # *** Must be run with root privileges *** cd /tmp wget http://www.rarlab.com/rar/unrar-linux-x64.gz gunzip -N unrar-linux-x64.gz cp /tmp/unrar /usr/bin chmod 0755 /usr/bin/unrar echo "Unrar installed." {% endmark -%}
